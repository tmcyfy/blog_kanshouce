    • macOS播放器不完全手册

    MPV是 macOS 最近几年评价最高的播放软件，没有之一。
    在MPV的网站上，TITLE是这样写的：

    a free, open source, and cross-platform media player

    支持多平台，官方下载戳这里 , GITHUB传送门戳这里

    安装

    在MACos中，也支持命令行安装：

    $ brew tap mpv-player/mpv  
$ brew install mpv

    配置

    下面主要列出如何正确的在macOS上配置MPV，达到最好的播放效果。
    如果采用APP的直接下载方式安装，需要创建MPV的配置文件，否则MPV将会按照默认的播放形式进行播放。 

    $ ~/.config/mpv/mpv.conf

    这意味着如果你没有这个文件，需要创建一个。
    我在这个文件中的配置是这样的： 

    screenshot-template=~/Desktop/mpv-screenshot-%f-%p //截图路径
screenshot-format=png  //文件名格式和图片格式
no-border  //去掉播放视频时播放器丑陋的边框
autofit-larger=100%  //设置了播放器最大宽高为屏幕大小，不会改变视频宽高比。避免当视频文件分辨率比屏幕分辨率还大时屏幕显示不全画面
sub-auto=fuzzy //设置了自动加载字幕文件的方式，默认是自动加载跟视频文件名完全相同（后缀除外），但是有些字幕文件名类似 xxxx.chs.srt 或 xxxx.eng.srt，设置 sub-auto=fuzzy，只要包含了视频文件名的字幕文件就会被自动加载
vo=opengl-hq:interpolation:icc-profile-auto  //最后一行配置参考自在Mac上看片的正确姿势。

    其它提示

    MPV不吃srt不吃srt不吃srt，所以下到srt字幕的，需要将其转换为ass格式。
    如果你的 GPU 不是太旧（HD4000 或以上），可以使用高质量 OpenGL 渲染 

    vo=opengl-hq:icc-profile-auto  //如果你是在刷新率为 60 Hz 的显示器上观看 24 fps 的片子，可以启用 interpolation 选项减少镜头平移时的顿挫感       
vo=opengl-hq:interpolation:icc-profile-auto
