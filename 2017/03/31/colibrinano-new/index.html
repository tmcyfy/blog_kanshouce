    • 新产品：10 KHZ至500 MHZ直接采样接收器 - COLIBRINANO

    ColibriNANO是一个新的SDR产品，目前处于预购阶段，预计以准备在2017年4月底交付销售。

    ColibriNANO是一个直接采样接收器（没有调谐器），它可以接收来自在过采样模式下为10 kHz至500 MHz，标准模式为10 kHz至55 MHz的信号。使用14位ADC，可提供高达110 dB的阻塞动态范围，并且可以以高达3 MHz的采样速率运行。

    ColibriNANO公示资料：

    ColibriNANO显着特点包括阻塞动态范围（BDR）〜110dB，原始0.01-55 MHz覆盖，过采样模式下高达500 MHz，低通滤波器（LPF）<60 MHz，与HDSDR，与传统SDR＃和ExpertSDR2完全兼容软件。

    支持的平台是适用于HTML5浏览器的Windows®XP-10，Linux和Web客户端。

    远程操作（TCP / IP）接口是内置的，并为业余，商业和政府应用提供即插即用解决方案。

    官方地址：https://www.nsiradio.com

    目前ColibriNANO 在nsiradio.com网站上售价为$249.95。

    在一些网站的推荐资料（软文）是这样写的：

    通过新的ColibriNANO，您将能够以许多不同的方式享受LF，MW和Shortwave聆听。例如，您可以使用1.5 MHz采样率记录整个中波段，使用CW分离器对CW进行解码，通过将ColibriNANO插入到我们的RPI服务器中进行远程控制。这种小型SDR的应用范围无限。所有这一切在一个小的U盘！

    ColibriNANO配有德州仪器ADS4145 14位直接采样ADC和内置低55 MHz通滤波器，可以旁路以接收高达500 MHz的信号。

    CW撇渣器和撇渣器使用外部ExtIO库，ColibriNANO可与第三方软件（如HDSDR）一起使用。

    这不是在目前发现的便宜的USB加密狗，这款高品质的SDR接收器是由Expert Electronics开发的，具有坚固的铝底盘，ESD保护，USB 2.0接口和优质的SMA天线连接器。

    最重要的是，ColibriNANO旅行在你的口袋里，只需要你的电脑和天线！它的终极便携式SDR接收器！

    配置：

    • 接收带宽：0.1 - 55 MHz
    • 采样接收：0.1 - 500 MHz
    • 阻塞动态范围（BDR）：110 dB
    • 灵敏度：20微米波段0.05 uV，前置放大器= 0
    • IMD3动态范围：95 dB
    • ADC分辨率：14位@ 122.88 MHz
    • 采样率：48,96,192,384,768kHz和1.5,3.0MHz
    • IQ分辨率：24位（采样率为1.5和3 MHz时为16位）
    • 射频输入：（SMA连接器，高达15kV ESD保护）
    • 前置放大器范围：从31.5到+6 dB，步长为0.5 dB
    • 工作温度：-10℃ ～ 60℃
    • 尺寸：90х25х17mm
    • 重量：0.043kg

    看起来这个接收机可能会与即将到来的Airspy HF +有所竞争。Airspy HF +声称类似规格，包括0 - 270 MHz的频率范围，14位ADC和108 dB阻塞动态范围。但HF +的目标价格低于$200。
