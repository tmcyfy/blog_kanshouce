    • WELLE.IO：适用于WINDOWS / LINUX的新型RTL-SDR＆AIRSPY DAB / DAB +解码器

    Albrecht Lohofener在Github上发布了一个名为“welle.io”的新软件包，它是一款免费的DABs和DAB +解码器和播放器，支持RTL-SDR（直接或通过rtl_tcp）和Airspy软件定义的收音机。该软件可以在Windows和Linux上运行，并且还支持Raspberry Pi 2/3和便宜的Windows 10平板电脑。

    Albrecht在软件介绍里写到，他的软件是qt-dab代码库的一个分支，开发目标是创建一个易于使用的DAB + DAB +软件接收器。该软件仍在大力发展，Albrecht提到他正在寻找开发人员和测试人员来帮助改进软件并报告错误。

    我自豪地介绍一个新的开源DAB / DAB +接收应用程序 welle.io https://www.welle.io。welle.io是qt-dab http://github.com/JvanKatwijk/qt-dab（旧dab-rpi和sdr-j-dab） 的分支， 目的是开发一个易于使用的DAB / DAB +接收应用程序。它 支持高DPI和触摸显示，甚至在便宜的电脑上运行， 如Raspberry Pi 2/3和100€中国Windows 10平板电脑。作为输入 设备，welle.io支持rtlsdr和airspy。

    目前Windows二进制构建都可以在项目GitHub上找到。对于Linux和Raspberry Pi用户，则需要从源代码编译代码，但将来他计划提供Ubuntu快照。

    我们给了welle.io软件一个简短的测试，顺利的跑了起来，但是很遗憾，在深圳没有找到任何DAB信号。

    软件自身有一个自动频道扫描功能可扫描所有可能的DAB频道和高级模式，以查看技术信息，如频率，SNR和错误率。该软件还具有良好的触摸屏和友好的GUI，可自动下载并显示DAB / DAB +程序指南信息。
